# Worlds mobile skeleton

Expo SDK 54 prototype for the Worlds hackathon MVP, compatible with the public Expo Go 54 app on iPhone and Android.

## Run locally

```bash
npm install
npx expo start --go
```

Scan the terminal QR code with Expo Go on an iPhone connected to the same network.

## Included flows

- Home and curated Worlds library
- World detail and pre-run setup
- Route-first active run with a pause sheet
- Personalized story recap
- Journey timeline
- Runner profile, history, and settings

All values are illustrative. GPS, audio, motion sensing, accounts, and persistence are intentionally not implemented yet.
